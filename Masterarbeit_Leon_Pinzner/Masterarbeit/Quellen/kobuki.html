New in groovy

Documentation

Components of this stack:

  • kobuki_auto_docking : Automatic docking for Kobuki

  • kobuki_bumper2pc : Publish bumpers and cliff sensors events as a pointcloud so navistack can use them

  • kobuki_controller_tutorial : Tutorial-related code

  • kobuki_description : URDF and Gazebo model description of Kobuki

  • kobuki_keyop : Keyboard teleoperation for Kobuki

  • kobuki_node : A ROS node wrapper for the kobuki driver

  • kobuki_random_walker : Random walker demo controller for Kobuki

  • kobuki_safety_controller : Watches the bumper, cliff and wheel drop sensor to allow safe operation

  • kobuki_testsuite : Set of tools to thoroughly test Kobuki's hardware
